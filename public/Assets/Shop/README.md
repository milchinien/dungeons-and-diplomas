# Shop Assets

Diese Dateien werden für das Shop-Rendering benötigt.

## Benötigte Dateien

| Dateiname | Größe | Beschreibung |
|-----------|-------|--------------|
| `sign.png` | 128x64 px | Shop-Schild mit "SHOP" Text |
| `counter.png` | 64x64 px | Holztresen-Tile |
| `door_locked.png` | 64x64 px | Geschlossene Shop-Tür |
| `door_open.png` | 64x64 px | Offene Shop-Tür |

## Design-Hinweise

- Pixel-Art im Dungeon-Stil
- 64px Tile-Größe
- PNG mit Transparenz
- Dunkle Fantasy-Ästhetik

## Erstellung

Tools: Piskel, Aseprite, oder GIMP
Siehe: `/public/Assets/Items/Icons/ICON_CREATION_GUIDE.md`

## Fallback

Ohne diese Assets werden einfache Platzhalter gerendert:
- Schild: Braunes Rechteck mit goldenem "SHOP"-Text
- Tresen: Braune Holzfläche
